# URL Shortener API

A complete URL shortener service built with Python, FastAPI, and SQLite.

## Features

- ✅ Shorten URLs with optional custom codes
- ✅ Redirect to original URLs
- ✅ Click tracking and analytics
- ✅ URL expiration
- ✅ Soft delete (deactivate)
- ✅ Web UI for easy use
- ✅ RESTful API
- ✅ SQLite database (lightweight)
